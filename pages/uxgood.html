There are many ways to fix the pre-filled text issue. Below, I have proposed one method of fixing the issues. This style can be modified to clarify many additional questions throughout the survey. As a result, the survey will offer a much better user experience.



This design improves the user experience because the user does not have to delete any text. The user can automatically add the numbers into the textbox without deleting the pre-existing numbers. If there is an error, the user can easily reset the field. If the user wants to confirm the number they selected, the submit button allows an easy route to confirm the selection.
